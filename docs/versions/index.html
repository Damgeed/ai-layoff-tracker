Versioning Policy

The AI Layoff Tracker dataset follows semantic versioning (SemVer):

  • Major versions (v2.0, v3.0…): Schema changes, reclassification sweeps, or substantial restructuring. Breaking changes to downstream consumers.
  • Minor versions (v1.1, v1.2…): New entries added to the dataset. No breaking schema changes.
  • Patch versions (v1.0.1, v1.0.2…): Corrections to existing entries. Data fixes only.

Each release is accompanied by a changelog entry. For individual entry-level changes between releases, see the Entry History page. For corrections applied to specific entries, see the Corrections page.
