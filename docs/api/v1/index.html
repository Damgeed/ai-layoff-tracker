API v1 Reference

Version 1 of the AI Layoff Tracker public API. All endpoints are read-only, require no authentication, and serve data under a CC0 public domain dedication.

Base URL

https://ailayofftracker.com

All endpoints return Content-Type: application/json unless noted. CORS is enabled. UTF-8 throughout.

Rate Limits

This API is served from GitHub Pages (static files). There are no server-side rate limits — but please be reasonable. If you plan to hit endpoints at high frequency, download the full dataset once and work locally:

curl -O https://ailayofftracker.com/api/entries.json  # ~200KB gzipped

Attribution

The dataset is CC0 — attribution is not required. However, we request:

  • Cite as: "AI Layoff Tracker, retrieved [DATE], https://ailayofftracker.com"
  • If you publish research using this data, let us know so we can reference your work
  • Link back to ailayofftracker.com when using in articles

Endpoints

Dataset

GET /api/entries.json Full dataset — all verified entries with evidence, sources, and classification 
curl https://ailayofftracker.com/api/entries.json | jq '.entries | length'  # 27
GET /api/entries.csv Downloadable CSV — Excel, Sheets, pandas compatible 
import pandas as pd
df = pd.read_csv("https://ailayofftracker.com/api/entries.csv")
print(f"{len(df)} entries, {df['jobs_lost'].sum():,} jobs")
GET /api/feed.xml RSS 2.0 feed — all entries with full metadata

Statistics

GET /api/stats.json Computed statistics — totals, breakdowns, trends GET /api/confidence.json Confidence methodology — scoring rules per entry

Per-Entity

GET /api/company/{slug}.json Per-company data (27 available) 
# List all company slugs
curl -s https://ailayofftracker.com/api/entries.json | jq -r '.entries[].slug' | sort -u
GET /api/industry/{slug}.json Per-industry data (8 available) GET /api/country/{slug}.json Per-country data (5 available)

Transparency

GET /api/changelog.json Dataset changelog and version history GET /api/corrections-schema.json Corrections submission format GET /api/weekly-validation-report.json Automated data quality report (9 checks)

Entry Schema

Each entry in /api/entries.json has this structure:

FieldTypeDescription
idstringUnique identifier (e.g. "klarna-2024-05")
companystringCompany name
slugstringURL-safe identifier (e.g. "klarna")
datestringISO date (YYYY-MM-DD)
countrystringCountry where layoffs occurred
industrystringIndustry classification
jobs_lostintegerNumber of jobs affected
impact_percentinteger% of company workforce affected
classificationstringOne of: DIRECT_AI_REPLACEMENT, AI_DRIVEN_RESTRUCTURING, AI_REALLOCATION, MARKET_DISRUPTION
confidence_scoreinteger0-100 based on evidence quality
summarystringHuman-readable summary of the event
evidencearrayEvidence items [{type, description}]
sourceobject{url, title, publisher, archive_url}
ceo_quotestring|nullCEO statement mentioning AI (if available)

Code Examples

Python

import requests, json

# Fetch full dataset
data = requests.get("https://ailayofftracker.com/api/entries.json").json()
entries = data["entries"]

# Filter: Tier 1 only
tier1 = [e for e in entries if e["classification"] == "DIRECT_AI_REPLACEMENT"]
print(f"Tier 1 (Direct AI Replacement): {sum(e['jobs_lost'] for e in tier1):,} jobs")

# Stats endpoint
stats = requests.get("https://ailayofftracker.com/api/stats.json").json()
print(f"Total: {stats['total_jobs_lost']:,} | Monthly rate: {stats['monthly_rate']:,}")

JavaScript

const resp = await fetch("https://ailayofftracker.com/api/entries.json");
const data = await resp.json();

// Group by industry
const byIndustry = {};
data.entries.forEach(e => {
  byIndustry[e.industry] = (byIndustry[e.industry] || 0) + e.jobs_lost;
});
console.log(byIndustry);

curl + jq

# Top 5 companies by jobs lost
curl -s https://ailayofftracker.com/api/entries.json | \\
  jq '[.entries | group_by(.company)[] | {company: .[0].company, jobs: (map(.jobs_lost) | add)}] | sort_by(-.jobs) | .[0:5]'
