API

Structured access to the AI Layoff Tracker dataset. All endpoints are public, no authentication required.

Dataset

GET /api/entries.json Full dataset — all 27 verified entries with evidence, sources, and confidence breakdowns GET /api/entries.csv Downloadable CSV — import into Excel, Sheets, or data analysis tools GET /api/feed.xml RSS feed — subscribe for new entry notifications

Statistics

GET /api/stats.json Computed statistics — totals, breakdowns by classification, industry, country GET /api/confidence.json Confidence methodology — scoring rules and per-entry breakdowns GET /api/weekly-validation-report.json Weekly validation report — automated quality checks (9 checks)

Per-Entity

GET /api/company/{slug}.json Per-company data — all entries for a single company (27 available) GET /api/industry/{slug}.json Per-industry data — all entries in a sector (8 available) GET /api/country/{slug}.json Per-country data — all entries in a country (5 available)

Transparency

GET /api/changelog.json Changelog — dataset version history and change log GET /api/corrections-schema.json Corrections schema — how to submit corrections

Usage Notes

  • No API key required — all endpoints are public
  • CORS enabled — fetch from any domain
  • Content-Type: application/json (except CSV and RSS)
  • UTF-8 encoding throughout
  • Dataset updated as new verified entries are added
